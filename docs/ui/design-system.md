# Design System - HSG HIS

## Objetivo
Padronizar a interface do sistema hospitalar para consistência visual e usabilidade.

## Stack UI
- JSF
- PrimeFaces
- CSS customizado (override leve)

## Princípios
- Interface funcional (sistema hospitalar)
- Alta densidade de informação
- Baixa distração visual
- Foco em produtividade do usuário
- Responsividade para diferentes dispositivos
